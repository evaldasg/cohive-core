# Cohive::Core

An implementation of the [clean architecture](http://blog.groupbuddies.com/posts/20-clean-architecture).

## Installation

Add this line to your application's Gemfile:

    gem 'cohive-core'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cohive-core
